# Module 2: Digital Twin Simulation

## Overview
Physics simulation and environment building with Gazebo and Unity. Simulating sensors like LiDAR and cameras. High-fidelity rendering for human-robot interaction.

## Topics Covered
- Physics simulation with Gazebo
- Unity for high-fidelity visualization
- Sensor simulation: LiDAR, IMU, Depth, Cameras
- Collision, gravity, and rigid-body physics

For detailed specification, see the [full specification](../specs/4-digital-twin-simulation/spec.md).
