# Module 1: The Robotic Nervous System (ROS 2)

## Overview
Middleware for robot control with ROS 2 Nodes, Topics, and Services. Bridging Python Agents to ROS controllers using rclpy. Understanding URDF (Unified Robot Description Format) for humanoids.

## Topics Covered
- ROS 2 Nodes, Topics, Services, and Actions
- rclpy: Connecting Python agents to robotic controllers
- URDF for humanoid robot structure
- Launch files and parameter management

For detailed specification, see the [full specification](../specs/3-ros2-nervous-system/spec.md).
