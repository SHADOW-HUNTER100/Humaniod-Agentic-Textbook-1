# Module 3: AI-Robot Brain (NVIDIA Isaac)

## Overview
Advanced perception and training with NVIDIA Isaac Sim. Hardware-accelerated VSLAM and navigation with Nav2 for humanoid movement.

## Module Structure
This module is organized into the following chapters:

1. [Chapter 1: Introduction to AI Robot Brains](./module-3-ai-robot-brain/chapter-1-introduction-to-ai-robot-brains.md) - Understanding cognitive architectures for robots
2. [Chapter 2: Perception Systems](./module-3-ai-robot-brain/chapter-2-perception-systems.md) - Processing sensory data for environment understanding
3. [Chapter 3: Planning and Decision Making](./module-3-ai-robot-brain/chapter-3-planning-and-decision-making.md) - Algorithms for action selection and path planning
4. [Chapter 4: Learning and Adaptation](./module-3-ai-robot-brain/chapter-4-learning-and-adaptation.md) - Machine learning techniques for robotic systems

## Topics Covered
- Isaac Sim: Photorealistic simulation
- Isaac ROS: GPU-powered VSLAM and perception
- Nav2 for bipedal humanoid navigation
- Synthetic data generation
